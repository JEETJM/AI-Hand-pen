## 🌟 About the Project

**AI Hand Pen** is a real-time hand gesture drawing web application that lets users draw on the screen using their **index finger** through **webcam-based hand tracking**.

This project uses **MediaPipe Hands** to detect hand landmarks and track finger movement smoothly.  
When drawing mode is enabled, the application creates a glowing trail effect on the screen, making it feel like you are drawing in the air.

It is simple, interactive, creative, and runs directly in the browser without any installation.

## ✨ Features

- 🎯 Real-time hand tracking using **MediaPipe Hands**

🚀 How to Run Locally
1. Clone the repository
git clone https://github.com/JEETJM/AI-Hand-pen.git
2. Open the project folder
cd AI-Hand-pen
3. Run the project

Open index.html in your browser.

4. Allow webcam access

When the browser asks for camera permission, click Allow.

5. Start drawing

Show your hand to the camera

Move your index finger

Hold Shift to draw

Press Space to clear

🔍 How It Works

The application works in the following way:

The webcam captures live video input.

MediaPipe Hands detects the hand and its landmarks.

The tip of the index finger is tracked in real time.

Finger coordinates are mapped onto the canvas.

Smoothed points are drawn to create a glowing trail.

The user presses Shift to activate drawing.

Pressing Space clears all previous strokes.
